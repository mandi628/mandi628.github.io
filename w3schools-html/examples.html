Attributes

Elements can be further defined or styled with attributes. For example, adding the "title" attribute to a paragraph can make a pop-up tooltip.

You can use attributes to format images, as well. The 

width=" "
and 
height=" "
attributes set the dimensions of an image, within the element tag.
